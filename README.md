# TS

此项目仅用于初次学习 TypeScript 使用.
主要是对开发者重塑'类型思维',更好地从 JavaScript 中理解面相对象,以及更好理解强类型语言的静态类型检测.


## 涉及技术
Webpack TypeScript


## 项目运行

```
// 下载或克隆值至本地
git clone https://github.com/XiaoYimi/TS.git

// 进入项目 
cd TS

// 安装依赖
npm i

// 启动命令-开发环境
npm run start

// 启动命令-生产环境
npm run build

// 访问 
http://localhost:8080
```
